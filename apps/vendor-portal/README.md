# Vendor Portal - Личный кабинет адвоката/нотариуса

Next.js приложение для управления заявками, расписанием и данными завещаний.

## Установка

```bash
npm install
```

## Запуск

```bash
# Development
npm run dev

# Production build
npm run build
npm start
```

Приложение будет доступно на порту 3002.

## Структура проекта

```
src/
├── app/                    # Next.js App Router страницы
│   ├── appointments/       # Управление заявками
│   ├── clients/            # Список клиентов
│   ├── schedule/           # Управление расписанием
│   ├── settings/           # Настройки профиля
│   └── wills/              # Работа с завещаниями
├── components/             # React компоненты
│   ├── appointments/       # Компоненты заявок
│   ├── clients/           # Компоненты клиентов
│   ├── layout/            # Layout компоненты
│   ├── schedule/          # Компоненты расписания
│   ├── settings/          # Компоненты настроек
│   ├── shared/            # Общие компоненты
│   ├── ui/                # UI компоненты
│   └── wills/             # Компоненты завещаний
└── lib/                   # Утилиты и API клиент
    └── api.ts             # API клиент для работы с бэкендом
```

## Основной функционал

### Приоритет 1: Настройки профиля и расписания
- ✅ Настройка радиуса поиска офиса
- ✅ Настройка выезда на дом
- ✅ Управление расписанием (рабочие часы по дням недели)

### Приоритет 2: Управление заявками
- ✅ Список заявок с фильтрацией и поиском
- ✅ Детальный просмотр заявки
- ✅ Подтверждение/отклонение заявок
- ✅ Внесение данных завещания после встречи

### Приоритет 3: Уведомления и клиенты
- ✅ Уведомление о смерти клиента
- ✅ Список клиентов адвоката

## API интеграция

Все API методы находятся в `src/lib/api.ts`. Приложение использует JWT токен для аутентификации (хранится в localStorage).

## Переменные окружения

Создайте файл `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:3001
```
