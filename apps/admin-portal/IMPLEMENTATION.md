## Структура файлов

### Страницы (12 файлов)
1. `/auth/login` - Вход в систему
2. `/dashboard` - Дашборд
3. `/lawyer-notary` - Список адвокатов
4. `/lawyer-notary/create` - Создание профиля
5. `/lawyer-notary/[id]` - Детальный просмотр
6. `/lawyer-notary/[id]/edit` - Редактирование
7. `/lawyer-notary/statistics` - Статистика по адвокатам
8. `/wills/appointments` - Список заявок
9. `/wills/data/[id]` - Данные завещания
10. `/wills/executions` - Исполняемые завещания
11. `/` - Главная (редирект)
12. `/layout` - Общий layout

### Компоненты (7 файлов)
- `Button`, `Card`, `Input` - UI компоненты
- `StatusBadge`, `LicenseTypeBadge` - Бейджи
- `ExportButton` - Кнопка экспорта
- `ErrorDisplay` - Отображение ошибок

### API клиенты (2 файла)
- `lawyer-notary.ts` - API для адвокатов
- `wills.ts` - API для завещаний

### Утилиты (4 файла)
- `auth.ts` - Аутентификация
- `utils.ts` - Общие утилиты
- `export.ts` - Экспорт в CSV
- `useAdminAuth.ts` - Хук для проверки прав

### Конфигурация
- `package.json` - Зависимости
- `tsconfig.json` - TypeScript конфигурация
- `next.config.js` - Next.js конфигурация
- `tailwind.config.js` - Tailwind конфигурация
- `postcss.config.js` - PostCSS конфигурация
- `middleware.ts` - Middleware для Next.js

## Функциональность

### Управление адвокатами
- Просмотр списка с фильтрацией по статусу и типу лицензии
- Поиск по email, ФИО, номеру лицензии
- Создание, редактирование, удаление профилей
- Изменение статуса с уведомлениями
- Экспорт данных в CSV

### Заявки и завещания
- Просмотр всех заявок с фильтрацией
- Детальный просмотр данных завещания
- Просмотр исполняемых завещаний
- Экспорт заявок в CSV

### Статистика
- Общая статистика на дашборде
- Детальная статистика по каждому адвокату
- Метрики: заявки, завершенные завещания, рейтинг

### Безопасность
- Аутентификация через JWT
- Защита роутов
- Проверка роли ADMIN

## Запуск

```bash
cd apps/admin-portal
npm install
npm run dev
```

Приложение доступно на `http://localhost:3003`
