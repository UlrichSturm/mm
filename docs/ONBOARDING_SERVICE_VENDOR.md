## Overview

This guide helps service vendors (providing services like funeral planning, transportation, catering) join the Memento Mori platform.

### What Services You Can Offer

- Funeral planning services
- Transportation services
- Catering services
- Cleaning services
- Other funeral-related services

---

## Registration Process

### Step 1: Apply

1. Contact platform admin
2. Submit business information
3. Wait for approval

### Step 2: Account Creation

1. Admin creates your vendor account
2. You receive email with login credentials
3. Complete profile setup

---

## Profile Setup

### Required Information

- **Business Information:**
  - Business name
  - Business registration
  - Tax ID
  - Business address
  - Contact information

- **Service Information:**
  - Service types
  - Service areas
  - Service radius
  - Pricing structure
  - Availability

- **Operational Information:**
  - Working hours
  - Team size
  - Equipment/resources
  - Certifications/licenses

## Service Listing

### Creating Service Listings

1. **Go to Services Page:**
   - Navigate to Vendor Portal
   - Click "Create Service"

2. **Fill Service Details:**
   - Service name
   - Description
   - Category
   - Price
   - Duration
   - Images

3. **Set Availability:**
   - Service radius
   - Home visit availability
   - Working hours
   - Booking requirements

4. **Submit for Review:**
   - Admin reviews listing
   - Approval/rejection

### Service Requirements

- **Clear Descriptions:** Detailed service information
- **Accurate Pricing:** Transparent pricing
- **Professional Images:** Service-related photos
- **Complete Information:** All required fields

## Schedule Management

### Setting Up Schedule

1. **Working Hours:**
   - Set daily working hours
   - Configure time slots
   - Set availability

2. **Blocked Dates:**
   - Mark unavailable dates
   - Set holidays
   - Block personal time

3. **Service Radius:**
   - Set service delivery radius
   - Configure home visit settings
   - Update coverage areas

### Managing Appointments

- **View Appointments:** See all scheduled appointments
- **Accept/Reject:** Manage appointment requests
- **Reschedule:** Reschedule appointments if needed
- **Complete:** Mark appointments as completed

## Order Management

### Processing Service Orders

1. **View Orders:**
   - Go to Orders page
   - See all service orders
   - Filter by status

2. **Update Order Status:**
   - Confirm order
   - Mark as in progress
   - Mark as completed
   - Handle cancellations

3. **Client Communication:**
   - Contact clients
   - Provide updates
   - Resolve issues

### Service Delivery

- **On-Time Service:** Deliver services on schedule
- **Quality Service:** Maintain service quality
- **Professional Conduct:** Professional behavior
- **Documentation:** Keep service records
