## üéØ Usage Instructions

### For Pitch Deck:

1. Open `en/01_PITCH_DECK.md` or `ru/01_PITCH_DECK_RU.md`
2. Create PowerPoint/Keynote presentation
3. Use content as slide text
4. Add visual elements:
   - Charts for financial data
   - Icons for features
   - Images for team/product
   - Graphs for market data
5. Design professionally
6. Export as PDF
7. Practice presentation (10-15 minutes)

### For Business Plan:

1. Open `en/02_BUSINESS_PLAN.md` or `ru/02_BUSINESS_PLAN_RU.md`
2. Format in Word/Google Docs
3. Add:
   - Table of contents
   - Page numbers
   - Headers/footers
   - Charts and graphs
   - Professional formatting
4. Export as PDF
5. Consider German translation

### For Financial Model:

1. Open `en/03_FINANCIAL_MODEL.md` or `ru/03_FINANCIAL_MODEL_RU.md`
2. Create Excel workbook with 8 sheets:
   - Assumptions
   - Revenue Model
   - Cost Model
   - Cash Flow
   - P&L Statement
   - Balance Sheet
   - Scenarios
   - Key Metrics
3. Implement formulas
4. Add charts
5. Create summary PDF
6. Validate all calculations

### For One-Pager:

1. Open `en/04_ONE_PAGER.md` or `ru/04_ONE_PAGER_RU.md`
2. Design in Canva/InDesign/Illustrator
3. Add:
   - Company logo
   - Visual elements
   - Charts/graphs
   - Icons
4. Keep to 1 page
5. Export as PDF (high resolution)
6. Create both print and digital versions

### For Team Description:

1. Open `en/05_TEAM_DESCRIPTION.md` or `ru/05_TEAM_DESCRIPTION_RU.md`
2. Add professional photos
3. Format in Word/InDesign
4. Add:
   - Company logo
   - Team photos
   - Professional layout
5. Export as PDF
6. Consider German translation
