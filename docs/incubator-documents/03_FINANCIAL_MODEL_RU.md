## Модель дохода

### Источники дохода

#### 1. Комиссия с транзакций (Основной)

**Расчет:**
```
Доход = Транзакции × Средняя стоимость транзакции × Ставка комиссии
```

**Год 1:**
- Транзакции: 4,000-6,000
- Средняя транзакция: €5,500
- Комиссия: 12%
- Доход: €2.64M - €3.96M

**Год 2:**
- Транзакции: 15,000-25,000
- Средняя транзакция: €6,000
- Комиссия: 14%
- Доход: €12.6M - €21M

**Год 3:**
- Транзакции: 40,000-60,000
- Средняя транзакция: €6,500
- Комиссия: 16%
- Доход: €41.6M - €62.4M

#### 2. Премиум-услуги (Будущее)

**Год 2:**

**Год 3:**
- Премиум-подписки: 10% поставщиков

#### 3. Партнерства с юридическими услугами

**Год 1:**

## Сценарии

### Оптимистичный сценарий

**Допущения:**
- Более быстрое принятие рынком (0.15% → 0.8% → 1.5%)
- Более высокие стоимости транзакций (€6,000 → €7,000 → €8,000)
- Более низкие затраты (эффективные операции)

**Результаты:**
- Доход Год 1: €4.5M
- Доход Год 2: €25M
- Доход Год 3: €75M
- Безубыточность: Месяц 8

### Реалистичный сценарий

**Допущения:**
- Стандартное принятие рынком (0.1% → 0.5% → 1.0%)
- Стандартные стоимости транзакций (€5,500 → €6,000 → €6,500)
- Стандартные затраты

**Результаты:**
- Доход Год 1: €3M
- Доход Год 2: €17M
- Доход Год 3: €50M
- Безубыточность: Месяц 10-12

### Пессимистичный сценарий

**Допущения:**
- Более медленное принятие рынком (0.05% → 0.3% → 0.7%)
- Более низкие стоимости транзакций (€5,000 → €5,500 → €6,000)

**Результаты:**

## Инструкции для Excel модели

### Создание Excel модели

1. **Создайте 8 листов** как описано выше
2. **Настройте лист допущений** со всеми входными значениями
3. **Свяжите формулы** между листами
4. **Используйте именованные диапазоны** для ключевых допущений
5. **Добавьте проверку данных** для входных данных
6. **Создайте графики** для визуализации
7. **Добавьте инструменты анализа сценариев**
8. **Отформатируйте профессионально** для презентации

### Ключевые формулы

**Расчет дохода:**
```
=Транзакции × СредняяТранзакция × СтавкаКомиссии
```

**Расчет затрат:**
```
=ПостоянныеЗатраты + (Доход × СтавкаПеременныхЗатрат)
```

**Денежный поток:**
```
=ОперационныйДенежныйПоток - ИнвестиционныйДенежныйПоток + ФинансовыйДенежныйПоток
```

**Безубыточность:**
```
=ПостоянныеЗатраты / (1 - СтавкаПеременныхЗатрат)
```
