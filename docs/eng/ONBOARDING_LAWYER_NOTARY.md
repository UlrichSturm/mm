## Overview

This guide helps lawyers and notaries join the Memento Mori platform to provide will certification and execution services.

### What You Can Do

- Certify will documents
- Execute wills
- Manage appointments
- Track will executions
- Earn income from will services

---

## Registration Process

### Step 1: Create Account

1. Go to Admin Portal: http://localhost:3003
2. Admin creates your profile
3. You receive email with login credentials

### Step 2: First Login

1. Go to login page
2. Enter email and password
3. Complete profile setup

---

## Profile Setup

### Required Information

- **Personal Information:**
  - Full name
  - Email address
  - Phone number
  - Address

- **Professional Information:**
  - License type (Lawyer/Notary/Both)
  - License number
  - License expiration date
  - Years of experience
  - Specializations

- **Business Information:**
  - Office address
  - Service radius
  - Working hours
  - Languages spoken

### Profile Verification

- Upload license documents
- Verify contact information
- Complete background check
- Wait for admin approval

---

## Verification

### Verification Steps

1. **Document Upload:**
   - Professional license
   - ID verification
   - Business registration (if applicable)

2. **Admin Review:**
   - Admin reviews documents
   - Background check
   - Approval/rejection

3. **Status Updates:**
   - PENDING: Under review
   - APPROVED: Active on platform
   - REJECTED: Application denied

## Getting Started

### After Approval

1. **Complete Profile:**
   - Add professional photo
   - Write bio
   - Set availability

2. **Configure Settings:**
   - Set service radius
   - Configure working hours
   - Set appointment preferences

3. **Start Accepting Appointments:**
   - View appointment requests
   - Accept/reject requests
   - Schedule appointments

### Managing Appointments

- **View Appointments:**
  - Go to Appointments page
  - See all appointment requests
  - Filter by status

- **Accept Appointments:**
  - Review client information
  - Check availability
  - Accept appointment

- **Complete Appointments:**
  - Mark appointment as completed
  - Upload documents
  - Update will status

## FAQ

### How do I get approved?

Complete your profile, upload required documents, and wait for admin review (typically 1-3 business days).

### How do I get paid?

Payment is processed through the platform. You'll receive payment after completing will certification/execution.

### Can I set my own prices?

Currently, will service is free for clients. Future pricing models may allow custom pricing.

### How do I manage my schedule?

Use the Schedule page to set working hours, block unavailable dates, and manage appointments.
