# Epic 10: Client App - Catalog

**Phase:** MVP  
**Portal:** Client App  
**Module:** Catalog  
**Epic ID:** E-010

---

## Описание

Каталог услуг и товаров в клиентском приложении. Включает просмотр каталога, базовый поиск, фильтры и детальные страницы услуг.

---

## Функциональность

- Главная страница с каталогом услуг
- Просмотр списка услуг
- Пагинация результатов
- Базовый поиск по названию
- Фильтрация по категориям (базовая)
- Детальные страницы услуг
- Добавление в корзину (базовое)

**Исключено из MVP:**
- Геопоиск на карте
- Продвинутые фильтры
- Сортировка
- Сохраненные поиски

---

**Предшествующие эпики:**

## User Stories (примеры)

1. **Как Client**, я хочу просмотреть каталог услуг, чтобы найти нужную услугу
2. **Как Client**, я хочу искать услуги по названию, чтобы быстро найти нужное
3. **Как Client**, я хочу фильтровать услуги по категориям, чтобы сузить поиск
4. **Как Client**, я хочу видеть детали услуги, чтобы принять решение

## Технические детали

**Технологии:**
- Next.js 14
- TypeScript
- Tailwind CSS
- React Query / SWR (data fetching)

**Страницы:**
- `/` - Главная страница с каталогом
- `/services` - Каталог услуг
- `/services/:id` - Детали услуги
- `/services/search` - Поиск услуг

**API Integration:**
- `GET /services` - Список услуг (с пагинацией и фильтрами)
- `GET /services/:id` - Детали услуги
- `GET /categories` - Список категорий
