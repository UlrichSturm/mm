# Epic 16: Admin Portal - Authentication

**Phase:** MVP  
**Portal:** Admin Portal  
**Module:** Authentication  
**Epic ID:** E-016

---

## Описание

Реализация аутентификации в административном портале. Включает страницу логина администратора и интеграцию с Backend Auth API.

---

## Функциональность

- Страница логина (Admin)
- Защита роутов (только ADMIN роль)
- Logout функциональность
- Сохранение токенов

**Исключено из MVP:**
- Регистрация администраторов (создаются вручную)
- Password reset

---

- Epic 1 должен быть завершен (Backend API готов)

**Блокирует:**
- Epic 17: Admin Portal - Moderation
- Epic 18: Admin Portal - Dashboard
