# Epic 18: Admin Portal - Dashboard

**Phase:** MVP  
**Portal:** Admin Portal  
**Module:** Dashboard  
**Epic ID:** E-018

---

## Описание

Дашборд администратора с базовой статистикой платформы. Включает обзор ключевых метрик и быстрый доступ к функциям.

---

## Функциональность

- Базовая статистика (количество пользователей, заказов, услуг)
- Графики (базовые)
- Быстрый доступ к функциям
- Обзор платформы

**Исключено из MVP:**
- Расширенная аналитика
- Сложные отчеты
- Экспорт данных

---
