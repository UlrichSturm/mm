# Epic 17: Admin Portal - Moderation

**Phase:** MVP  
**Portal:** Admin Portal  
**Module:** Moderation  
**Epic ID:** E-017

---

## Описание

Модерация контента в административном портале. Включает модерацию поставщиков и услуг.

---

## Функциональность

- Модерация поставщиков (одобрение/отклонение)
- Модерация услуг (одобрение/отклонение)
- Список на модерацию
- Фильтрация по статусам
- Комментарии к модерации

**Исключено из MVP:**
- Модерация отзывов
- Автоматическая модерация
- История модерации

## Технические детали

**API Integration:**
- `GET /admin/vendors` - Список поставщиков
- `PATCH /admin/vendors/:id/status` - Изменение статуса
- `GET /admin/services` - Список услуг
- `PATCH /admin/services/:id/status` - Изменение статуса
