# Epic 33: Memorials

**Phase:** Phase 3  
**Portal:** Backend  
**Module:** MemorialsModule  
**Epic ID:** E-033

---

## Описание

Мемориальные страницы для усопших. Включает создание мемориальных страниц, публичные страницы и управление контентом.

---

## Функциональность

- Создание мемориальных страниц
- Публичные мемориальные страницы
- Управление контентом (фото, текст)
- Комментарии и соболезнования (опционально)

**Исключено из Phase 3:**
- Сложные шаблоны
- Видео контент
